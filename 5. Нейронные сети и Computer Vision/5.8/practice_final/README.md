## Как получилось это решение?

1. Нашла подходящую модель для высокоточной сегментации с открытым кодом и готовыми весами.
2. Увидела, что существует не только Colab demo, но и demo app на Hugging Face. Склонировала его.
3. Заменила чекпоинт на более общий (`isnet.pth` -> `isnet-general-use.pth`), убрала лишние объекты.
4. Добавила место для загрузки второго изображения в качестве фона, дописала вызываемую функцию, чтобы она накладывала фон и возвращала готовое изображение.
