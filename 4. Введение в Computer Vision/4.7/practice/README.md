# Распознавание рукописных символов EMNIST

## 1. Описание решения
Этот проект представляет собой реализацию REST API для 
распознавания рукописных символов, основанного на модели CatBoost. 
API использует модель, обученную на датасете EMNIST, для классификации 
изображений рукописных символов.
Гиперпараметры модели:     
	iterations=2500,  # Количество итераций обучения
	learning_rate=0.1,  # Скорость обучения	
	depth=6,  # Глубина дерева
	l2_leaf_reg=3,  # L2-регуляризация листьев
	eval_metric='Accuracy',
	task_type="GPU",  # Использование GPU
	random_seed=42,  # Случайное состояние
	# Параметры защиты от переобучения
	early_stopping_rounds=100,  # Прекращение обучения, если точность на валидационном наборе не улучшается 100 итераций
	use_best_model=True,  # Использование наилучшей модели по точности на валидационном наборе
	bagging_temperature=0.5  # Bagging для уменьшения корреляции деревьев
Метрика Accuracy=0.8340 на тестовых данных

Проект включает в себя следующие файлы:

main.py: Файл с кодом приложения FastAPI.
model.py: Файл с классом Model, который загружает обученную модель CatBoost и предоставляет метод для предсказания символа.
requirements.txt: Файл с перечнем зависимостей проекта.
emnist-balanced-mapping.txt: Файл с сопоставлением индексов классов и соответствующих символов.
model.pkl: Файл с обученной моделью CatBoost .


## 2. Установка и запуск сервиса
Инструкция по запуску
Установка зависимостей:
pip install -r requirements.txt

Запуск приложения:
uvicorn main:app --host 0.0.0.0 --port 8000

Доступ к API:
Откройте браузер и перейдите по адресу http://localhost:8000
Вы можете использовать инструмент Postman или curl для отправки запросов к API.

1. Клонирование репозитория:
git clone https://gitlab.skillbox.ru/anton_rokhmistrov/ml-advanced.git

2. Переход в директорию проекта:
cd <название вашего репозитория>

3. Сборка Docker-образа:
docker build -t <имя вашего образа> .

4. Запуск Docker-контейнера:
docker run -p 8000:8000 <имя вашего образа>
